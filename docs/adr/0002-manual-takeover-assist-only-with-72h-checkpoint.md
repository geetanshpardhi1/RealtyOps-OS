# Manual Takeover Uses Assist-Only Draft Mode with 72-Hour Checkpoint

When a Brokerage Agent manually takes over a lead in RealtyOps OS V1, AI autonomous sending is paused and AI remains available only as a drafting assistant. We also set a 72-hour checkpoint to explicitly decide whether to resume automation or continue manual handling. This decision protects human ownership during sensitive interactions while retaining AI leverage and preventing indefinite ambiguous control states.
